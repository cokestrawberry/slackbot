# Deferred — 이슈 검색 기능

> **등록일:** 2026-05-12
> **우선순위:** 높음
> **요청자:** 공식 봇 비교 분석

## 요구사항

`@지라봇 검색 로그인` → DB에서 키워드 검색하여 매칭되는 이슈 목록 반환.

## 예시

```
사용자: @지라봇 검색 로그인
봇:
🔍 "로그인" 검색 결과 (3건)
  • SLAC-7 로그인 페이지 /auth/login 500 에러 (진행 중, SP 2)
  • SLAC-11 로그인 세션 만료 처리 (해야 할 일, SP 3)
  • SLAC-15 로그인 UI 리뉴얼 (완료, SP 5)
```

## 구현 계획

- Haiku intent `search`는 이미 분류 가능 (현재 "준비 중" 안내만 반환)
- `IssueRepository.findBySummaryContaining` 재활용
- 키워드 매칭: `@지라봇 검색 <키워드>` 또는 Haiku가 `extracted.keyword` 추출
- 결과 포맷팅: Jira 링크 + 상태 + SP 포함
- 난이도: 낮음 (DB 쿼리 + 포맷팅만 추가)
