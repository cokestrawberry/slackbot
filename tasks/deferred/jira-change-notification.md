# Deferred — Jira 이슈 변경 알림

> **등록일:** 2026-05-12
> **우선순위:** 높음
> **요청자:** 공식 봇 비교 분석

## 요구사항

Jira에서 이슈 상태가 변경되면 (생성/수정/전환/댓글) Slack 채널에 실시간 알림을 보낸다.

## 구현 방법 (택 1)

### A. Jira Webhook
- Jira 프로젝트 설정 → Webhook 등록 → ngrok URL로 전달
- 장점: 실시간
- 단점: ngrok 무료 플랜은 URL 변경됨, 운영 시 고정 URL 필요

### B. 주기적 폴링
- `@Scheduled`로 N분마다 Jira 변경사항 조회 → 변경 있으면 Slack 알림
- 장점: 별도 설정 불필요
- 단점: 실시간 아님 (지연 있음)

## 알림 형태 예시

```
🔄 SLAC-7 로그인 500 에러
상태: 진행 중 → 완료
담당자: 김영현
```
